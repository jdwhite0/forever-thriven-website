# FOREVER THRIVEN WEBSITE

**What this is:** Next.js website for Forever Thriven, LLC — a family-owned group home in St. Petersburg, FL  
**Domain:** forever-thriven.com  
**Status:** V1 Built — Ready for GitHub + Vercel deploy

---

## Quick Links

- **Run locally:** `npm run dev` → http://localhost:3000
- **Build:** `npm run build`
- **Deploy:** Push to GitHub → Vercel auto-deploys

---

## Pages
